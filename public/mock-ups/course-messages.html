  • חזרה על שיעור 1 - מסמך תכנון, מדריך react & css
  • הקמת פרויקט חדש - שימוש ב github, vite, material ui

בחלק האחרון של השיעור יהיה זמן לתרגול וכן הצגת פניות.

  • עלינו לאחר הקורס מצגת התקנות נדרשות. מומלץ לבצע לפני תחילת השיעור. ניתן גם לבצע בזמן התרגול.
  • אשמח לראות את אחת מערכות התכנון וידוי עליה בכיתה. צורך שמעוניין להציג את העבודה (גם אם לא שלמה) - שלחו לי בקשת הודעה עם קובץ העבודה.
